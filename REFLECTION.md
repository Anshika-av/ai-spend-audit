# Reflection

This project taught me how full-stack applications are built and deployed in production environments.

The most valuable lesson was understanding how frontend systems interact with backend infrastructure, databases, permissions, and deployment platforms.

I learned:
- React state management
- Supabase integration
- database permissions and RLS
- asynchronous UI handling
- production deployment with Vercel

The most difficult issue involved debugging failed Supabase inserts caused by Row Level Security configuration errors.

This project improved both my frontend engineering and product-thinking skills.
